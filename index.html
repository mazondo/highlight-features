Highlight Features

An easy to use jQuery plugin for highlighting features and providing contextual help

Demo Github

Features

  • Add contextual help to your page with a single line of JS
  • Launch the script for the entire page or a single container
  • Easy and intuitive for users, just mouse over the blinking dots

How to use

Using the library is easy, just add a data-feature attribute to any html element on your page and run $("body").highlightFeatures("show");

Basic Example

    <a href="https://github.com/mazondo" data-feature="Your feature text goes here">Link text</a>

Works on any element

    <span data-feature="Works with spans as well">Span text</span>

    <div data-feature="And divs">div content</div>

Then run it

    <script>
      $(".help-link").click(function(e) {
        e.preventDefault();
        $("body").highlightFeatures("show");
      });
    </script>
